CAF2014: a dynamic model for Coffee Agroforestry Systems

QUICK START:

(1) Install gfortran on your computer.
(2) Install R on your computer.
(3) Install RStudio on your computer.
(4) In RStudio choose the 32-bit R version (-> Tools -> Global Options -> R General).
(5) Clone the CAF2014 directories to your computer.
(6) Open the file "INTRODUCTION_EXAMPLES_CAF2014.R" in RStudio and execute the code examples.
